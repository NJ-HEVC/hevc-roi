Initialize a YUV_Capture instance. Allocates memory used for internal processing. After initialization, frames can be read using YUV_read.

Parameter
[in]finThe file to read from. If the function succeeds, the capture will own this pointer from that point onwards.
[in]wThe frame width, in pixels.
[in]hThe frame height, in pixels.
[out]outThe instance to initialize.
enum YUV_ReturnValue YUV_read ( struct YUV_Capture *  cap)

Read a single frame from a previously-instantiated YUV_Capture instance.

Parameter
[in,out]capThe capture to read from. The result frame is stored in cap-ycrcb.
Rückgabe
YUV_OK if it's OK to keep reading. YUV_EOF if EOF was reached and no further reading is possible. The capture should be cleaned up.
